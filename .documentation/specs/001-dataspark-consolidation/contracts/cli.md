# CLI Contract: DataSpark Console

**Branch**: `001-dataspark-consolidation` | **Date**: 2026-03-30

## Overview

The DataSpark CLI (`dataspark`) provides command-line access to database discovery, export, schema analysis, and C# DTO generation. Built on `System.CommandLine 2.x`.

## Global Options

| Option | Type | Description |
|--------|------|-------------|
| `--help`, `-h` | flag | Display help for any command |
| `--version` | flag | Display version |
| `--verbosity` | enum | Logging level: `quiet`, `normal`, `detailed` (default: `normal`) |

## Commands

### dataspark discover

Discover SQLite databases in a directory.

```
dataspark discover --path <directory> [--recursive] [--format <text|json|markdown>]
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--path`, `-p` | string | Required | Directory to scan |
| `--recursive`, `-r` | flag | false | Include subdirectories |
| `--format`, `-f` | enum | text | Output format |

**Output (text)**:
```
Found 3 SQLite databases:

  Path                          Size      Tables
  ────────────────────────────  ────────  ──────
  ./data/chinook.db             864 KB    11
  ./data/northwind.db           1.2 MB    13
  ./data/sakila.db              3.4 MB    16
```

**Output (json)**:
```json
{
  "databases": [
    {
      "path": "./data/chinook.db",
      "sizeBytes": 884736,
      "tableCount": 11
    }
  ]
}
```

**Exit codes**: 0 = success, 1 = path not found, 2 = no databases found

### dataspark export

Export database tables to CSV.

```
dataspark export --path <database> --output <directory> [--tables <t1,t2>] [--delimiter <char>] [--no-headers]
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--path`, `-p` | string | Required | SQLite database path |
| `--output`, `-o` | string | Required | Output directory |
| `--tables`, `-t` | string | all | Comma-separated table names |
| `--delimiter`, `-d` | string | "," | CSV delimiter |
| `--no-headers` | flag | false | Omit column headers |

**Output**:
```
Exporting chinook.db...

  Table          Rows    Size     Status
  ─────────────  ──────  ───────  ───────
  artists        275     12 KB    ✓
  albums         347     18 KB    ✓
  tracks         3503    256 KB   ✓

Exported 3 tables (4125 rows) to ./csv/
```

**Exit codes**: 0 = all exported, 1 = database not found, 2 = some tables failed

### dataspark schema

Display database schema.

```
dataspark schema --path <database> [--format <text|json|markdown>] [--table <name>]
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--path`, `-p` | string | Required | SQLite database path |
| `--format`, `-f` | enum | text | Output format |
| `--table` | string | null | Specific table (null = all) |

**Output (markdown)**:
```markdown
# Schema: chinook.db (864 KB)

## artists (275 rows)

| Column   | Type    | Nullable | PK  |
|----------|---------|----------|-----|
| ArtistId | INTEGER | No       | Yes |
| Name     | TEXT    | Yes      | No  |
```

**Exit codes**: 0 = success, 1 = database not found

### dataspark generate

Generate C# DTOs from database schema.

```
dataspark generate --path <database> [--output <directory>] [--namespace <ns>] [--table <name>]
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--path`, `-p` | string | Required | SQLite database path |
| `--output`, `-o` | string | stdout | Output directory (creates one `.cs` per table) |
| `--namespace`, `-n` | string | "DataSpark.Models" | C# namespace |
| `--table` | string | null | Specific table (null = all) |

**Output (file generated)**:
```csharp
// Auto-generated by DataSpark CLI
namespace DataSpark.Models;

/// <summary>
/// DTO for the 'artists' table.
/// </summary>
public record Artist
{
    public long ArtistId { get; init; }
    public string? Name { get; init; }
}
```

**Type mappings**:

| SQLite Type | C# Type |
|-------------|---------|
| INTEGER | long |
| TEXT | string |
| REAL | double |
| BLOB | byte[] |
| NUMERIC | decimal |
| (nullable) | appends `?` |

**Exit codes**: 0 = success, 1 = database not found
